This setup guide will help you integrate the AI Proofreader Extension from scratch into your application.

### Install the extension

<Callout title="Subscription required" variant="warning">

This extension requires a valid Entry, Business or Enterprise subscription. To install the extension you need access to our [private registry](/guides/pro-extensions), set this up first.

</Callout>

Once done, you can install the extension from our private registry:

```bash
npm install @tiptap-pro/extension-ai-proofreader
```

### Import the extension in your editor

Add the `AiProofreader` to the list of extensions.

```ts
import { useEditor } from '@tiptap/react'
import StarterKit from '@tiptap/starter-kit'
import AiProofreader from '@tiptap-pro/extension-ai-proofreader'

// Initialize the editor
const editor = useEditor({
  extensions: [
    StarterKit,
    AiProofreader.configure({
      // Define proofreader rules
      rules: [
        {
          id: '1',
          title: 'Spell Check',
          prompt: 'Identify and correct any spelling mistakes',
          color: '#DC143C',
          backgroundColor: 'FFE6E6',
        },
      ],
      // … other options
    }),
    // … more extensions
  ],
  // Recommended: disable spellcheck to avoid conflicts
  editorProps: {
    attributes: {
      spellcheck: false,
    },
  },
})
```

### Decide how suggestions will be generated. Use our Content AI Cloud or your own backend and LLMs

The next step is to configure a way to generate the suggestions that will be displayed in the editor.

The easiest way to get started with the AI Proofreader is to use Tiptap Content AI Cloud. Our API will generate suggestions from your rules and content. Follow this setup guide: [Use Tiptap Content AI Cloud](/content-ai/extensions/ai-proofreader/use-with-content-ai-cloud)

You can also use your own backend and LLMs to generate proofreading suggestions. You can find a guide here: [Integrate your own backend and LLMs](/content-ai/extensions/ai-proofreader/integrate-your-own-llms)

### Customize the user interface

Finally, you’ll need to configure the editor styles so that the suggestions are displayed in the UI.

By default, the AI Proofreader suggestions have CSS classes and color attributes to help you style them. Use these CSS styles to show a colored underline below the suggestions.

```css
.tiptap-ai-proofreader-suggestion {
  border-bottom: 2px solid var(--tiptap-ai-proofreader-suggestion-color);
  margin-bottom: -2px;
}

.tiptap-ai-proofreader-suggestion--selected {
  background-color: var(--tiptap-ai-proofreader-suggestion-background-color);
  transition: background-color 0.5s;
}
```

You can customize the suggestions' appearance even more by providing custom decorations. To show a tooltip when you select a suggestion, follow this guide: [Show a tooltip when you select a suggestion](/content-ai/extensions/ai-proofreader/features/display-suggestions-in-the-editor)
